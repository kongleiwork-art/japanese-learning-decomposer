# Output Format

Use Chinese for explanations unless the user asks otherwise.

## Single Word Or Phrase

```markdown
# 日语学习拆解

## 原文
……

## 读音和意思
- 读音：
- 词性/类型：
- 常见意思：
- 这里更自然的理解：

## 用法提示
- 常见搭配：
- 语气/场景：
```

End with:

```text
需要我把这份内容直接做成带日语全文朗读的网页阅读器 HTML 吗？
```
